# zabbix-template
This a template for zabbix on azure by using arm moudle.

重要说明：
     1、此模板内的zabbix入口为 http://IP/zabbix or  http://dns/zabbix  (<IP/DNS>为zabbix monitor的IP地址或者dns地址）
     2、 此模板的部署要在新portal上 http://portal.azure.cn 或者使用powershell部署，详情可参考该链接  https://www.azure.cn/documentation/articles/open-source-azure-resource-manager-zabbix
